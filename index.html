• AutoOutline : Pdf Structuring

As part of Adobe India's national hackathon, developed a fully containerized solution that automatically extracts structured data from complex PDF documents and converts it into clean, schema-compliant JSON outputs. The system efficiently identifies titles, hierarchical headings (H1, H2, etc.), and page numbers while supporting multilingual text, including non-Latin scripts. Designed to work entirely offline, the pipeline adheres to strict resource constraints (CPU-only, 8 cores, 16GB RAM) and leverages pdfminer.six for robust layout analysis. The solution was Dockerized to ensure portability, ease of deployment, and complete reproducibility across platforms. Key Contributions: - Designed and implemented process_pdfs.py for parsing PDF files with structured output. - Built a Docker container for isolated, offline processing of documents. - Handled layout, font styles, and heading detection for structured outline generation. - Ensured compatibility with multilingual and complex script PDFs. Tech Stack: - Python, pdfminer.six, Docker, Linux/Windows (amd64)

    PersonaMap: Multi-Collection PDF Analyzer

    A scalable solution for analyzing multiple PDF document collections based on persona-driven use cases. The system takes in real-world task descriptions ("jobs-to-be-done") along with PDFs, extracts relevant content, and ranks them using semantic similarity models. For example, given the task “Plan a 4-day trip for college friends” and travel-related PDFs, the tool identifies the most relevant sections (e.g., on cuisine, transportation) tailored to a Travel Planner persona. The pipeline performs structured extraction , contextual understanding using sentence-transformers, and outputs clean JSON files per collection. The entire system supports both local and Docker-based execution, with automatic directory-based collection handling. Tech Stack: - Languages & Frameworks: Python - Libraries: pdfminer.six, sentence-transformers, torch - Containerization: Docker (amd64) - Execution Modes: Local & Docker-based - Output: JSON with ranked content relevant to personas
